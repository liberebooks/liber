**The ebook library you'd build today if you started from zero.**

Modern ebook library manager and converter — fast, beautiful, local-first.

## About

Liber is a spiritual successor to Calibre — same problem space, completely different take on UX and aesthetics.

- Import EPUB, MOBI, PDF, and more
- Auto-fetch cover art and metadata
- Convert to Kindle, Kobo, and other device formats
- Built-in EPUB and PDF viewer
- Project Gutenberg and Standard Ebooks browser
- Reading stats and progress tracking
- 100% local, no accounts, no cloud required

> **Note:** This repository contains compiled releases only.
> Source code is maintained in a private development repository.

Liber checks for updates automatically. You can also trigger a manual check from **Settings → About & Updates**.
